# Volume-Scroll
Chrome and Firefox extension for changing video volume by scrolling


### The project was built using
*   **Operating System:** Windows 11.
*   **Node.js Version:** v21.2.0
*   **Yarn Version:** v4.4.0

### Installation
To install the necessary dependencies, navigate to the root directory and run:

- yarn install

### Build Script
To build the extension, run:

- yarn build

### Output
The final extension files will be generated in the `dist/` folder.
